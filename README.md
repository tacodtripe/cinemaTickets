# Cinema Tickets Demo

CLI that simulates software used to sell cinema tickets and assign available seats.

S = available seat
B = occupied seat

## Live version

- [here](https://replit.com/@tacodtripe/CinemaTickets#Main.java)

## Built With

- Java
